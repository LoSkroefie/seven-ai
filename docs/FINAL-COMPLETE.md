## 🔧 TROUBLESHOOTING

### **Seven won't start:**
```bash
python preflight_check.py
```
Check what's missing, install dependencies.

### **GUI not showing:**
Make sure tkinter is installed (comes with Python).

### **No voice output:**
Check pyttsx3 installation:
```bash
pip install pyttsx3
```

### **Ollama errors:**
1. Make sure Ollama is running
2. Check model is installed:
   ```bash
   ollama list
   ```
3. Pull model if needed:
   ```bash
   ollama pull llama3.2
   ```

### **Vision not working:**
Install OpenCV:
```bash
pip install opencv-python
```

### **Performance issues:**
Check Metrics tab in GUI for:
- High response times
- Memory usage
- CPU usage
- Follow recommendations
